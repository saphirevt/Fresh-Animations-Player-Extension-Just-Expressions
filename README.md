# Fresh Animations: Player Extension + Just Expressions
This resource pack is a compatibility bridge that combines the player movements of Fresh Animations with the animated eye logic of Just Expressions.

Follow the instructions on the official Just Expressions [wiki](https://github.com/IthanMendoza/Fresh-Moves/wiki) to add animated eyes to your skin.

## Requirements
To function correctly, this pack requires the same dependencies as the originals:
- Entity Model Features (EMF)
- Entity Texture Features (ETF)
- Fresh Animations: Player Extension (The base animations).

## Installation Order
In your Resource Packs menu, make sure the order looks like this (from top to bottom):
1. This Compatibility Pack (Top priority)
2. Fresh Animations: Player Extension
3. Just Expressions (Optional, for textures)
